===
Work내용

* 프로그램이 시작되면 아래의 메뉴를 제공한다.

>메뉴 : 1.이름 입력 2.출력 3.종료

* 1을 입력하면 showInputDialog를 제공하여 이름을 받아 저장

* 2를 입력하고 Enter을 치면 showMessageDialog를 사용하여 아래와 같이 출력

	이름<br>
	ㅡㅡㅡ<br>
	제이슨<br>
	메튜<br>
	해피<br>

	확인을 누르면 닫히면서 메뉴 출력.

* 3을 입력하고 Enter을 치면 showConfirmDialog에서 확인을 누르면 감사합니다 출력하고 프로그램 종료

* 과제 완료
